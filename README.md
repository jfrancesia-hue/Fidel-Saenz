# Fidel Sáenz — Plataforma territorial de campaña

Demo premium estática para presentar una campaña territorial en San Fernando del Valle de Catamarca.

## Stack

- HTML estático: `index.html`
- CSS separado: `styles.css`
- JS separado: `script.js`
- Sin dependencias ni build.

## Qué incluye

- Hero institucional premium.
- Selector de vista: vecino / comando.
- Gestión en números con contadores animados.
- Ejes de propuesta.
- Mapa territorial SVG interactivo.
- Simulador de crecimiento.
- Formulario demo sin persistencia.
- Feed demo de adhesiones.
- Panel interno de comando con métricas ficticias.

## Cómo abrir

Abrir `index.html` directamente o servir el directorio con cualquier servidor estático.

## Importante

Los datos son ficticios y solo sirven para explicar la experiencia de campaña. Antes de publicar o mostrar políticamente, reemplazar textos, métricas y propuestas por contenido validado.
